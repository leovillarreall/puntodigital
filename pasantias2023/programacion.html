Nuestros Cursos

Enseña a programar de cero sitios web integrando los lenguajes de programación HTML y CSS. Entre los conceptos básicos, se profundiza en qué es HTML, cómo usarlo, cómo programar con CSS con HTML, cómo hacer un sitio web y cómo crear un formulario con HTML. Es recomendable realizar el curso desde una computadora con conexión a internet. Se enseñará a instalar y usar Sublime Text, el editor de texto con el que se generará el documento HTML. Tiene una duración de 40 horas y cuenta con certificado de aprobación.
